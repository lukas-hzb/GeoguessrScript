# GeoScript

A community-driven Geoguessr UserScript that displays meta hints and allows users to contribute new locations.

## Features
- **Live Hints**: Shows helpful metas for the current location.
- **Crowdsourced**: Data is loaded from `data/locations.json`.
- **Contribution**: Use the "Add Meta" button to generate JSON for new entries.

## Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/).
2. Install the script from this repository.
